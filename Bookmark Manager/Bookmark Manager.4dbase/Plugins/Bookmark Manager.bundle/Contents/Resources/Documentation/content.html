Bookmark Manager

Plugin Identifier: 20000

Summary

This plugin allows access to Mac OS bookmarks.

What are bookmarks?

Introduced in Mac OS X 10.6, bookmarks offer a convenient way of tracking down file system resources (files, folders, packages...) independent of their name and location. You can think of it as Mac OS aliases. Once a file system bookmark is stored in the database as BLOB, you can retrieve it's current location even if was moved, renamed or trashed in the meantime.

Bookmarks

BOOKMARK Create

bookmark:=BOOKMARK Create (path)

path (TEXT)

Path of the file system item to bookmark.

bookmark (BLOB)

Newly created bookmark.

Discussion

This command creates a bookmark of the given file system item.

BOOKMARK Export to file

error:=BOOKMARK Export to file (bookmark; path)

bookmark (BLOB)

The bookmark previously created with BOOKMARK Create.

path (TEXT)

The destination path to export the bookmark.

error (LONGINT)

0 if the operation was successful, otherwise a system error code (NSError).

See Foundation Constants Reference for possible values.

Discussion

This command exports the bookmark as an alias file.

BOOKMARK Resolve

path:=BOOKMARK Resolve (bookmark)

bookmark (BLOB)

The bookmark previous created with BOOKMARK Create.

path (TEXT)

The file system path of the bookmarked item.

Discussion

This command returns the path of a bookmarked item.
